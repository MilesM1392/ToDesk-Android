# 📱 To-Do School App (Android)

Une application Android moderne de gestion de devoirs permettant aux étudiants d’organiser et suivre leurs tâches quotidiennes de manière simple et efficace.

---

## ✨ Aperçu du projet

Cette application permet de consulter une liste de devoirs récupérée depuis une API, avec une interface fluide, animée et adaptée au mode sombre.

## 🎯 Objectif

Ce projet a été réalisé dans un cadre personnel afin de :

- Avoir une expérience simplifiée du site ToDesk (todesk.milesm.fr) pour un usage mobile
- Développer une application Android native complète
- Apprendre à consommer une API REST avec Retrofit
- Utiliser RecyclerView pour des listes dynamiques
- Améliorer l’expérience utilisateur avec des animations
- Structurer une application maintenable et évolutive

---

## 🚀 Fonctionnalités

### 📋 Gestion des devoirs
- Affichage dynamique des devoirs
- Chargement des données depuis une API
- Mise à jour automatique de la liste

### 🔄 Interactions
- Rafraîchissement de la liste
- Animation d’apparition des éléments (effet de descente)
- Interface fluide et réactive

### 🎨 Interface utilisateur
- Design moderne basé sur ConstraintLayout
- RecyclerView optimisé
- Support du **Dark Mode**
- UI simple, propre et intuitive

---

## 🧱 Architecture

- **Activities** : gestion des écrans
- **RecyclerView + Adapter** : affichage des listes
- **Retrofit** : communication avec l’API
- **JSON parsing** : conversion automatique des données

---

## 🔌 API

L’application utilise **Retrofit** pour :

- Envoyer des requêtes HTTP
- Récupérer des données au format JSON
- Transformer automatiquement les réponses en objets exploitables

---

## 🎞️ Animations

Les éléments de la liste apparaissent progressivement avec une animation de descente, améliorant l’expérience utilisateur et donnant un rendu plus moderne.

---

## 🌙 Dark Mode

Le mode sombre est intégré afin de :

- Améliorer le confort visuel
- Réduire la fatigue oculaire
- S’adapter aux préférences utilisateur

---

## 🛠️ Technologies utilisées

- Java (Android)
- Android SDK
- RecyclerView
- ConstraintLayout
- Retrofit
- XML (UI)
- Animations Android

## 🐛 Difficultés rencontrées

- Intégration de Retrofit

- Synchronisation des animations avec les données
- Mise en place du Dark Mode
